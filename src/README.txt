File Work IO
--------------

1) Створіть проект File-Work-IO-task,
де виконайте завдання.
+

2) У корні проекту створіть папку files,
для роботи з файлами. Базовий URL до цієї
папки вкажіть в окремому класі.
+

3) У класі FileWriteService, за допомогою
java.io.FileOutputStream, реалізуйте
створення та запис у файл txt формату.
+

4) У класі FileReadService, за допомогою
java.io.FileInputStream, реалізуйте
читання цього файлу.
+

5) Введення назви файлу (без розширення) та
його текстовий контент реалізуйте через
Scanner, у класі Main.
+

6) У класі Main створіть меню вибору дій
по роботі з файлом:
- створити та записати;
- читати.
+

7) Клас Main повинен виводити результат роботи
із файлом.
+
